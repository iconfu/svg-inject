SVGInject v2 Examples

All icons below are loaded as <img> elements and injected inline with SVGInject. View source to see how it works.

Basic injection

Just add onload="SVGInject(this)" to any <img>.

CSS currentColor

SVGs using fill="currentColor" inherit the CSS color property. Same SVG, three different colors — pure CSS.

Different sizes

Control size with CSS classes — the SVG scales naturally via viewBox.

Mouse interactions

CSS :hover works on injected SVGs — color, scale, glow, rotation, opacity. Try them all.

Effects

CSS animations, filters, and transforms — all pure CSS, no JavaScript needed.

Dark mode

Set color on a dark container — currentColor does the rest.

ID uniqueness

Same SVG with gradient IDs injected three times — no conflicts, each renders correctly.

Accessibility

Meaningful image with alt — gets role="img" and aria-label. Decorative image with alt="" — gets aria-hidden. Inspect the DOM to verify.

JS-driven injection (no onload)

These images have no onload — they're injected from JavaScript after DOM ready.
