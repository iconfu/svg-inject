SVGInject v2 Examples

Every icon below starts as a normal <img> and gets replaced with inline <svg> by SVGInject. Once inline, you can style them with pure CSS. View source to see the code.

Basic injection

Add onload="SVGInject(this)" to any <img>. The image becomes an inline SVG.

Change color with CSS

If the SVG uses fill="currentColor", you can change its color with the CSS color property. Same SVG file, different CSS class.

ID uniqueness

This SVG uses a <linearGradient> with id="grad1". Injected three times — SVGInject makes each ID unique so all three render correctly. Without this, only the first would show the gradient.

grad1--inject-1
grad1--inject-2
grad1--inject-3

Accessibility

SVGInject sets ARIA attributes automatically. Inspect the DOM to verify.

alt="Favorite icon"
→ role="img" aria-label="..."
alt="" (decorative)
→ role="none" aria-hidden="true"

JavaScript-driven injection

No onload attribute needed. Call SVGInject() from JavaScript on any <img> element or NodeList.
